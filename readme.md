# AI Repository

This repository aims to automate fixing of XPath errors in Test Automation


## Technologies Used

- Node.js
- Express.js
- WebDriverIO
- Bootstrap


## Getting Started

1. Clone the repository to your local machine.
2. Navigate to the root directory.
3. Install dependencies with `npm install`.
4. Start the server by running `node server.js`.
5. Open a new terminal tab and execute `npm run wdio` to run WebDriverIO tests.
6. To replicate XPath Error, change ID of any element in `register.html` and run `npm run wdio`
